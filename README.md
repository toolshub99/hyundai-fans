# Hyundai Website

A static website showcasing Hyundai vehicle models and information.

## Project Structure

- `public/`: Static HTML files and assets
- `js/`: JavaScript files
- `data/`: Data files (e.g., models.json)
- `public/css/`: Stylesheets
- `public/images/`: Image assets

## Setup

1. Clone the repository
2. Open `public/index.html` in a web browser

## Deployment

This project is configured for Cloudflare Workers. Use Wrangler to deploy:

```bash
wrangler deploy
```
